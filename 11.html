01

アニメ背景は「固有色」で塗る

BASE COLOR // 写真じゃなく絵

リアルなザラつきや反射を作り込む必要はありません。アニメ背景はそのモノが持つ「固有色」をフラットにのせるのが基本。光や影はあとで光のレッスンで足します。

NG ── 作り込みすぎ
写真のような細かい汚れや反射。アニメの中で浮いてしまい、線画の絵と馴染みません。
OK ── フラットな固有色
木は木の色、布は布の色を平らに一面。木目など最小限の柄だけ。絵としてスッと馴染みます。
02

Blenderの中で直接塗る

TEXTURE PAINT // 筆でのせる

まずはベースカラーを一色設定し、テクスチャペイントモードでモデルに直接筆を入れます。立体を見ながら塗れるので、塗り分けの位置を確認しやすい。

手順 ── 一色 → 筆で塗る
マテリアルにベースカラー → Texture Paint → 筆で塗り分け → F で筆サイズ
塗った絵は「画像テクスチャ」として保存。こまめに保存しないと消えるので注意。
▸
画面の使い分け
3Dビューで立体に直接、UVエディタで展開図の上から。どちらに塗っても同じ1枚の画像に反映されます。

03

Photoshopと行き来して描く

PS LINK // 得意な道具で

細かい絵は使い慣れた2Dソフトで描くのが一番速い。UVの「下書き」を書き出し、Photoshop等で描き込んで、画像を貼り直す往復ワークフローを覚えましょう。

WORKFLOW ── 往復の4ステップ
1 UVを書き出す ── UVエディタから「UV配置をエクスポート」。線だけの下書きPNGが出ます。
2 PSで描く ── 下書きを下のレイヤーに敷いて、その上から色と木目を描き込む。
3 画像で書き出す ── 下書きレイヤーを消して、PNGで保存。
4 Blenderに貼る ── マテリアルのベースカラーに画像を接続。立体にピタリと反映。
