01

家具を呼び込む ── アペンド

APPEND // 別ファイルから持ち込む

家具は別々のファイルに保存してあるはず。それを部屋のファイルに「アペンド」で読み込みます。コピーが取り込まれるので、元のファイルは汚れません。

手順 ── ファイル → アペンド
File ▸ Append → .blendを開く → Object の中の家具を選ぶ
アペンド=コピーを取り込む / リンク=参照だけ。今回は自由に動かしたいのでアペンド。
▸
置く時のコツ
スケール感を最初に合わせる。床にきちんと接地させ、余白(間)を残して配置すると画が締まります。
.blend
▸
呼び込む アペンド
02

アングルを決める ── カメラ

CAMERA // 絵コンテに合わせる

カメラを1台追加し、絵コンテのアングルに合わせます。視点をのぞきながら動かせる「カメラにビューをロック」が便利。覗いて、構図を整えます。

STEP
1
カメラを置いて 覗く
カメラを追加し 0 でカメラ視点へ。テンキー0で切り替えます。
STEP
2
ビューをロックして 構図決め
「ビューをロック」をオンにすると、視点操作がそのままカメラの動きに。覗きながら構図を追い込めます。
✦
構図のコツ
三分割の線を表示して、主役を交点に。少し低い/高いアングルで「らしさ」が出ます。
覗いて決める テンキー 0
03

印象を変える ── レンズ(焦点距離)

LENS // 広角 ⇄ 望遠

同じアングルでも焦点距離(mm)を変えると印象が激変。広角は広く奥行きが出てダイナミックに、望遠は奥が圧縮されて落ち着いた画に。

広角 ── 〜24mm
空間が広く・奥行き強調。狭い部屋でも全体が入る。歪みでダイナミックに。
望遠 ── 85mm〜
奥が圧縮され、落ち着いた静かな画に。主役が引き立つ。
✦
アニメ背景の定番
部屋の背景美術は標準〜やや広角(35〜50mm)が馴染みやすい。レンズはカメラデータのプロパティで設定します。
