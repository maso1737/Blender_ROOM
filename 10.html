01

UV展開ってなに?

UV // 紙の箱を開く

立体のままでは絵を塗れません。お菓子の箱をハサミで開いて平らにするのと同じ。その平らな図が「UV(展開図)」、塗るための画用紙です。

3D ── 立体のまま
面が立体的に曲がっていて、絵を「貼る場所」が決められない。このままでは塗れません。
UV ── 開いて平らに
全ての面が2Dの座標に並ぶ。ここに絵を描けば、立体にピタリと貼り付きます。
02

切れ目を入れて、開く

SEAM // どこで切るか

どの辺で切り開くかを「シーム」で指定します。目立たない裏側で切るのがコツ。指定したら展開(Unwrap)するだけ。

手順 ── 切って開く
辺を選択 → Ctrl+E シームをマーク → 全選択 → U 展開
UVエディタに展開図が出ます。シームは赤い線で表示されます。
▸
歪みをチェック
チェッカー模様を貼って確認。マス目が正方形に近いほど良い展開。伸びていたらシームを足します。
Ctrl+E → U
03

納品できるデータに整える

ASSET // プロのお作法

作って終わりではプロ失格。他人が開いても迷わないデータに整えて、ターンテーブル動画を添えて提出します。

CHECKLIST ── 提出前にこれだけ
✓ 名前を付ける ── "Cube.001" のままにしない。"chair_seat" など中身が分かる名前に。
✓ 原点を整える ── オブジェクトの原点を底面の中心へ。配置がラクに。
✓ スケールを適用 ── Ctrl+A で拡縮を 1.0 に戻す。
✓ コレクションで分類 ── 家具・小物などフォルダ分けして提出。
